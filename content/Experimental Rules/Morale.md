# Death of the Warlord
If a player's Warlord is slain, all other units belonging to that player receive a -1 on all Morale  tests for the rest of the game.  
# Wavering
Armies that have taken significant losses will begin to decline in morale. Each unit that is not retreating in an army that is Wavering takes a Morale test at the end of the Player turn. Units in a wavering army may only Rally on Heroic Morale  
## Objective-Based Morale
At the end of the Player turn, if a Wavering Army controls fewer objectives than the opponent, all Units in that Army receive -1 to Leadership for any Wavering tests for each objective that the opponent controls more than their own.
